부모 요소보다 자식 요소가 클 때 표시 설정

  • hidden: 넘친 부분의 자식요소 숨김
  • auto: 넘치면 스크롤을 생성
  • scroll: 자식의 크기에 상관없이 스크롤 생성
  • visible: 표시

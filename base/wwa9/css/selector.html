선택자

  • 태그(요소) 선택자 : 태그이름 사용
  • 아이디 선택자 : # 사용
  • 클래스 선택자 : . 사용
  • 전체 선택자 : * 사용
  • 후손(하위) 선택자 : 공백 사용
  • 자식 선택자 : > 사용
  • 형제 중 바로 아래 동생만(인접) 선택자 : + 사용
  • 형제 중 모든 동생들 선택자 : ~ 사용
  • 그룹 선택자 : , 사용
  • 속성 선택자 : [] 사용
    • [속성명] : 속성을 가지고 있는 요소
    • [속성명="속성값"] : 속성을 가지고 있고 값이 일치하는 요소
    • [속성명~="속성값"] : 속성을 가지고 있고 값을 단어로 포함하는 요소
    • [속성명|="속성값"] : 속성을 가지고 있고 값과 같거나 뒤에 -기호로 연결된 요소
    • [속성명^="속성값"] : 속성을 가지고 있고 값으로 시작하는 요소
    • [속성명$="속성값"] : 속성을 가지고 있고 값으로 끝나는 요소
    • [속성명*="속성값"] : 속성을 가지고 있고 값을 포함하는 요소
  • 가상클래스 선택자 : : 사용
    • a:link - 링크가 설정된 요소 선택자(a태그 전용)
    • a:visited - 링크에 연결된 주소로 방문한 적이 있는 요소 선택자(a태그 전용)
    • a:hover - 마우스를 올린 요소 선택자(모든 태그)
    • a:active - 클릭한 요소 선택자(모든 태그)
    • a:focus - 제어가 설정된 요소 선택자(a 태그, 인풋 태그)
  • 가상요소 선택자 : : 사용
    • ::first-letter : 문장의 첫 번째 문자
    • ::first-line : 문장의 첫 번째 줄
    • :first-child : 첫 번째 자식 요소
    • :last-child : 마지막 자식 요소
    • :nth-child(n) : n번째 자식 요소
    • ::before자식으로 첫 번째 요소를 삽입(닫기 태그가 없는 요소에는 사용 못함)
    • ::after자식으로 마지막 번째 요소를 삽입(닫기 태그가 없는 요소에는 사용 못함)

우선순위

일반적 규칙

밑에 있는 소스가 우선권을 가진다.

인라인스타일이 우선권을 가진다.

!important를 적용하면 우선권을 가진다.

우선순위 값

선택자 우선순위
전체 선택자(*) 0
태그 선택자(a, div...) 1
가상 선택자(:first-child,..) 10
클래스 선택자(.) 10
아이디 선택자(#) 100

한국인은 비타민 D가 부족한 사람이 많다.

비타민D는 햇빛에 있다.

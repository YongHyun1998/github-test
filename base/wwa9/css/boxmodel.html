박스 모델

넓이, 높이, (안, 바깥)여백, 테두리를 박스 모델이라 함.

보통 요소의 기본 넓이 계산은 넓이 + (안쪽 여백 * 2) + (테두리 두께 * 2)

box-sizing속성을 border-box로 설정하면 안쪽 여백과 테두리를 자동으로 계산함.

인풋 속성

속성 설명
checked 해당 radio, checkbox를 로딩 후 바로 선택
disabled 인풋 요소를 사용 못하게 함.
max number, range의 최고값 지정
maxlength 글입력 시 최고 글자 수 지정
min number, range의 최소값 지정
pattern 정규표현식 지정
readonly 읽기 전용 설정
required 반드시 지정해야 할 인풋 설정
size 크기 지정
step number, range의 증가/감소 값 지정
value 실제 값, 버튼 인풋은 버튼의 글자

cafe24에 비디오 등록 순서

서브라임 텍스트
  1. mp4파일을 준비(youtube convertor mp4로 검색)
  2. 로컬의 카페24 폴더로 mp4파일을 이동
  3. 서브라임 : 확장자를 html로 파일이름 변경(Rename...)
  4. 서브라임 : 동영상 파일을 업로드
  5. 서브라임 : 업로드 후 SFTP/STP > Rename Local and Remove files를 선택하여 mp4로 확장자 변경
비주얼 스튜디오 코드
  1. mp4파일을 준비(youtube convertor mp4로 검색)
  2. 로컬의 카페24 폴더로 mp4파일을 이동
  3. 코드 : 파일질라를 실행 후 카페24에 mp4를 저장 할 폴더 생성
  4. 코드 : 파일질라에서 왼쪽 부분(로컬)에서 확장자(mp4)를 html로 파일이름 변경
  5. 코드 : 파일질라에서 동영상 파일을 업로드
  6. 코드 : 파일질라에서 오른쪽, 왼쪽의 html 파일을 월래의 mp4로 확장자를 변경

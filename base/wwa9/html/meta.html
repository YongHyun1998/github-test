meta 태그

<meta charset="UTF-8"> : 문서의 문자셋 정의(파일의 포멧 종류 설정)

<meta name="keywords" content="HTML, CSS, JavaScript"> : 검색 엔진이 검색하는 검색어

<meta name="description" content="Free Web tutorials"> : 설명글

<meta name="author" content="John Doe"> : 제작자 이름

<meta http-equiv="refresh" content="30"> : 새로고침

<meta name="viewport" content="width=device-width,initial-scale=1.0> : 표시 장치의 종류에 상관없이 1:1로 표시할 수 있게 함.

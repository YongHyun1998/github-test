HTML 작성 관례

  1. 문서의 시작은 !DOCTYPE으로 한다.
  2. 태그는 소문자를 사용한다.
  3. 닫기 태그가 있는 태그는 정확하게 닫는다.
  4. 속성은 소문자로 사용한다.
  5. 속성의 값은 큰따옴표로 묶는다.
  6. 이미지 태그는 alt속성을 가진다.
  7. 속성의=사이는 띄워쓰지 않는다.
  8. html태그에는 lang속성을 사용한다.
  9. 태그 사이에는 이유없이 띄워쓰지 않는다.
  10. 정확한 들여쓰기를 한다. ctrl + [ : 내어쓰기, ctrl + ] : 들여쓰기

        <li>닫기 태그가 있는 태그는 정확하게 닫는다.</li>

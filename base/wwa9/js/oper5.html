비트 연산자

비트 연산자 설명
&(And) 1 & 1 = 1, 1 & 0 = 0, 0 & 0 = 0
2(0010) & 3(0011) = 2(0010)
|(Or) 1 | 1 = 1, 1 | 0 = 1, 0 & 0 = 0
2(0010) & 3(0011) = 3(0011)
~(Not) 0 => 1, 1 => 0
~2(0010) = 13(1101) / ~3(0011) = 12(1100)
^(XOR) 1 ^ 1 = 0, 1 ^ 0 = 1, 0 ^ 0 = 0
2(0010) ^ 3(0011) = 1(0001)
<<(Left Shift) 2(0010) << 2 = 8(1000)
>>(Right Shift) 2(0010) >> 2 = 0(0000)

변수와 상수

변수(var, let) : 데이터를 저장하는 장소

변수는 특수문자 중에서 _, $만 사용하고 숫자로 시작할 수 없음

카멜표기법(변수) : 두 번째 단어부터 첫자를 대문자로 사용

파스칼표기법(객체) : 첫자는 대문자로 사용

헝가리언표기법(변수) : 타입을 첫음에 붙임

let : 변수 선언 키워드

var과 let의 차이점

- var은 재선언이 가능하지만 let 재선언이 불가능

- var은 함수단위 실행, let 블록단위 실행

상수(const) : 변하지 않는 고정된 값(대문자만 사용)

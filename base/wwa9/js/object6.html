내장 객체 - Array

데이터값의 모음, 대괄호로 만듦

  • slice(x, y) - x번째부터 y이전까지의 새로운 배열을 반환
  • join(x) - 배열사이에 x를 삽입한 문자열 반환
  • concat(arr) - 배열 두개를 붙여 새로운 배열 반환
  • toString() - 배열 내용을 문자로 반환
  • shift() - 첫 번째 배열 삭제
  • unshift(val) - 배열의 앞에 val값을 추가
  • pop() - 마지막 배열 삭제
  • push(val) - 배열의 뒤에 val값을 추가
  • splice(start, delete, item, ...) - 배열을 수정 함
  • reverse() - 배열을 역순으로 정렬
  • sort() - 값의 오름차순, 내림차순으로 정렬
  • forEach() - 배열의 크기만큼 반복하며 특정 작업을 실행
  • map() - 배열의 크기만큼 반복하며 특정 작업 실행 후 새로운 배열을 반환
  • filter() - 조건에 맞는 새로운 배열 반환
